## 🚀 Featured Projects

### 🧪 Demo Examination Platform
Solves the problem of manual and static examination generation with a highly intelligent examination system, powered by AI, with three unique modes: Practice, Timed, and Survival. Uses OpenRouter to create MCQs for any topic in real-time and stores them in Supabase while managing user data and statistics locally.   
🔗 [Repo](https://github.com/Redietkebede/demo-examination-platform) · 🌐 [Live Demo](https://demo-examination-platform.vercel.app)

---

### 🏋️ Gym Booking System
Eases the pain associated with manually managing gym schedules by offering a full stack booking and management solution for elite fitness studios. Customers can view services and make bookings from a seamless user interface, whereas the administrative panel will allow users to manage bookings, services, and status updates, thanks to the use of Prisma ORM and PostgreSQL.  
🔗 [Repo](https://github.com/Redietkebede/gym_booking_system) · 🌐 [Live Demo](https://gym-booking-system-one.vercel.app)

---

### 🤖 AI Learning Path Generator Platform
Tackles the challenge of self-learning through unstructured means by creating personalized and structured roadmaps that guide learners based on their learning objectives and proficiency levels. Equipped with OpenRouter LLMs and Supabase storage, it also includes lead generation, event tracking, SEO-friendly blog routing, and an entire animated SaaS landing page.  
🔗 [Repo](https://github.com/Redietkebede/AI-Learning-Path-Generator-Platform) · 🌐 [Live Demo](https://ai-learning-path-generator-platform.vercel.app)

---

### 📊 Financial & Economic Intelligence Dashboard
Attempts to address the challenge of making sense of unprocessed data from the economy and markets by creating a comprehensive end-to-end intelligence system. This system collects macroeconomic indicators using the FRED API and market information using the Alpha Vantage API with a Python ETL process, calculates macro signals such as the Macro Risk Index and Currency Strength Ranking, and delivers them using a Next.js interface.  
 🔗 [Repo](https://github.com/Redietkebede/Financial-Economic-Intelligence-Dashboard) · 🌐 [Live Demo](https://financial-economic-intelligence-monitoring.vercel.app/)
